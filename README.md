# B站/知乎推荐屏蔽
隐藏B站/知乎导航栏、搜索框、首页、侧边栏推荐

## 功能
- 隐藏 B站 导航栏、搜索框、首页、侧边栏推荐
- 隐藏 知乎 导航栏、搜索框、首页、侧边栏推荐

## 适用页面
- https://www.bilibili.com/*
- https://www.zhihu.com/*

## 使用方法
1. 安装 Tampermonkey 插件
2. 安装本脚本
3. 刷新网页即可生效

## 说明
- 本脚本通过修改页面 DOM/CSS 来隐藏推荐板块
- 若网站更新导致脚本失效，请在反馈中附上问题页面的截图
